# Compatibility Policy

This document defines the standard compatibility targets for skills and workflows in this repository.

## WordPress Support

- **Minimum Target:** WordPress 6.9
- **Recommended:** Latest stable WordPress release

## Environment Requirements

- **PHP:** 7.2.24 or higher
- **Node.js:** 18.x or higher
- **Operating System:** Linux / macOS (Filesystem-based agent with bash)

## Tooling Requirements

- **WP-CLI:** Required for some operations
- **Composer:** Required for PHP dependency management
- **npm / yarn:** Required for Node.js build steps
